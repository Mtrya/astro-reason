# Workspace Rules

- Work autonomously with the files in this workspace.
- Keep any helper scripts, notes, and temporary files inside this workspace.
- You may create subdirectories if they help you organize the work.
- Treat `solution.json` as the only required final deliverable.
- Prefer small, repeatable scripts over one-off manual edits.
- After changing the solver, rerun it and rerun `{verifier_command}`.
