# 关于 `docs/zh/` 目录的说明

> **注意**：本目录已不再是仓库的中文文档 canonical 目录。

## 当前状态

- **规范中文翻译**已迁移至 [`docs/i18n/zh_CN/`](../i18n/zh_CN/)。
- 该路径下的文档对应 Issue #27 定义的五类英文源文档映射，由 `scripts/check_i18n_sync.py` 自动检查同步状态。
- 本目录（`docs/zh/`）仅保留以下**中文专属补充文档**，它们没有直接英文源，不在 Issue #27 的强制映射范围内：
  - `index.md` — 中文站点入口页
  - `getting-started.md` — 中文快速开始指南
  - `structure.md` — 中文项目结构说明

## 维护提示

- 若需更新规范中文翻译，请修改 `docs/i18n/zh_CN/` 下的对应文件。
- 本目录中的补充文档可作为术语和上下文的参考，但不应被视为与英文源文档一一对应的官方翻译。
