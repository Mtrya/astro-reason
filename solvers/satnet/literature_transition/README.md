# SatNet Transitional Literature-Backed Solver

This directory records a temporary solver-facing baseline for SatNet.

It is intentionally non-runnable. The entries in `metrics.yaml` capture
reference metrics generated from the upstream SatNet package and tracked
fixtures. They may be useful for reporting while real SatNet solvers are still
missing, but they must be presented separately from reproduced solver runs.

This baseline should be replaced when a repository-owned SatNet solver exists.
